+## Committing and pushing changes
+Follow these steps after you finish edits locally:
+
+1. Check what changed:
+   ```bash
+   git status
+   ```
+2. Review your modifications (optional but recommended):
+   ```bash
+   git diff
+   ```
+3. Stage the files you want to include:
+   ```bash
+   git add <file1> <file2>
+   ```
+   To stage everything, run `git add .`.
+4. Create a descriptive commit:
+   ```bash
+   git commit -m "Your message here"
+   ```
+5. Push to GitHub (replace `work` if you're on a different branch):
+   ```bash
+   git push origin work
+   ```
+6. If prompted for credentials, use your GitHub token (recommended) or SSH keys.
+
+After pushing, create or update the pull request on GitHub with a short summary and any test results.
